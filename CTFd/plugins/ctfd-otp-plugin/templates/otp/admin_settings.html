About OTP
What is OTP?

OTP (One-Time Password) is a time-based authentication method that provides an additional layer of security beyond just passwords. Users must have an authenticator app (like Google Authenticator, Authy, or Microsoft Authenticator) to generate these codes.

How does it work?
  1. Users set up OTP by scanning a QR code with their authenticator app
  2. The app generates a new 6-digit code every 30 seconds
  3. When OTP is required, users enter the current code from their app
Tip: Before requiring OTP for admin actions, make sure all administrators have set up OTP on their accounts. Otherwise, they may be locked out of sensitive functions.
